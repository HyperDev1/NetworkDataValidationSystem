<objective>
Create `.github/workflows/deploy.yml` — the GitHub Actions CI/CD pipeline that automates the full build → push → deploy cycle on every `main` branch push.

Behavior:
- **main push**: `build-and-push` job (auth → build → push to AR with sha + latest tags) + `deploy` job (gcloud run deploy with new image)
- **PR**: `build-check` job (auth → build only, no push, fast Dockerfile validation)

Purpose: Every `main` push automatically ships the latest code to Cloud Run without any manual intervention. PRs get instant feedback on whether the Dockerfile is broken before merge.
Output: `.github/workflows/deploy.yml`
</objective>

**Job 1: `build-and-push`** (runs only on push to main — use `if: github.event_name == 'push'`)
- `runs-on: ubuntu-latest`
- `permissions: { contents: read, id-token: write }` — id-token required for WIF
- Steps:
  1. `actions/checkout@v4`
  2. `google-github-actions/auth@v2` with `workload_identity_provider: ${{ secrets.WIF_PROVIDER }}` and `service_account: ${{ secrets.WIF_SERVICE_ACCOUNT }}`
  3. Configure Docker to push to Artifact Registry: `gcloud auth configure-docker us-central1-docker.pkg.dev --quiet`
  4. `docker/setup-buildx-action@v3` (enables BuildKit for cache support)
  5. `actions/cache@v4` for Docker layer cache:
     - key: `docker-${{ runner.os }}-${{ hashFiles('requirements.txt', 'Pipfile.lock') }}`
     - restore-keys: `docker-${{ runner.os }}-`
     - path: `/tmp/.buildx-cache`
  6. `docker/build-push-action@v5`:
     - `context: .`
     - `push: true`
     - `tags: |`
       `${{ env.IMAGE }}:sha-${{ github.sha }}`
       `${{ env.IMAGE }}:latest`
     - `cache-from: type=local,src=/tmp/.buildx-cache`
     - `cache-to: type=local,dest=/tmp/.buildx-cache-new,mode=max`
  7. Move cache (prevents cache bloat): `mv /tmp/.buildx-cache-new /tmp/.buildx-cache`
  - `outputs: image: ${{ env.IMAGE }}:sha-${{ github.sha }}` (expose for deploy job)
  - `timeout-minutes: 20`

**Job 2: `deploy`** (runs only on push to main)
- `needs: build-and-push`
- `runs-on: ubuntu-latest`
- `permissions: { contents: read, id-token: write }`
- Steps:
  1. `google-github-actions/auth@v2` (same WIF credentials)
  2. `google-github-actions/deploy-cloudrun@v2`:
     - `service: network-data-validation`
     - `region: us-central1`
     - `image: ${{ env.IMAGE }}:sha-${{ github.sha }}`
     - `flags: --no-allow-unauthenticated`
- `timeout-minutes: 10`

**Job 3: `build-check`** (runs only on pull_request)
- `if: github.event_name == 'pull_request'`
- `runs-on: ubuntu-latest`
- `permissions: { contents: read, id-token: write }`

     - No tags needed (not pushing)
     - `cache-from: type=gha` (GitHub Actions cache, simpler for PR checks)
     - `cache-to: type=gha,mode=max`
- `timeout-minutes: 15`
